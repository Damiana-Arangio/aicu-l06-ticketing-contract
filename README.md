# Materiali Studenti - Lezione 06

## Cosa Devi Produrre

Per il task:

```txt
Serve creare ticket dal supporto.
```

devi preparare:

- contract sketch leggero;
- 1 payload valido;
- 2 payload invalidi;
- risposta attesa per ogni payload;
- error model minimo;
- data sketch annotato;
- Mermaid leggero che visualizza solo campi e relazioni motivati.

## File Da Usare

```txt
template/contract-sketch-create-ticket.md
template/data-sketch-create-ticket.md
template/prompt-critica-contract.md
```

## Regola

Come visto in aula, riusa il percorso.

Non copiare campi, dati e comportamento del caso discusso.
